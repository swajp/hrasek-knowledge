# Hrasek Knowledge Base

Znalostni baze pro bota **Hrasek** na Discord serveru Mendelu.

Hrasek je AI asistent, ktery odpovida na otazky studentu. Cerpa z tohoto repozitare a z fora **Otazky a odpovedi** na Discordu.

## Jak to funguje

1. Studenti pouziji prikaz `/ask` na Discord serveru
2. Hrasek prohledava tento soubor `knowledge.md` a vlakna z fora
3. Na zaklade nalezenych informaci odpovi na otazku

## Jak prispet

Precti si [CONTRIBUTING.md](CONTRIBUTING.md) a posli Pull Request s upravou souboru `knowledge.md`.

Kazdy prispevek pomaha dalsim studentum!
